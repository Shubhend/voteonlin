Requirements:
==============
   * PHP Version 4 and above
   * Javascript should be enabled.
   * Mysql Version 4 and above

Features:
==========
a) Allows you to manage and administrate the survey.
b) Get ratings from employees using this survey script.
c) Simple, responsive and easy to integrate.
d) Allows you to add unlimited questions with multiple options.
e) You can block and unblock any questions and also view the statistics of each questions.


Installation:
==============
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation.

Extracting files:
 a)  Unzip survey-script.zip to extract the files and folders.

Configuring a Database:
 a)  Create a database manually eg: “survey”
 b)  In the browser execute the file survey-script/admin/install.php
 c)  Provide the database details like Username, Password, DBname, and Hostname.
 d)  This should install the product.
 e)  For Security reasons, please delete install.php from the server.

File Permission:
 Set read, write permission to the file survey-script/admin/auth/config.php

Admin Login:
 a) Go to the page survey-script/admin/index.php
 b) Use the admin user and password to login.
 c) From here you can manage categories, questions and answers.


How to use Review System Script:
=================================

Add Poll :

a) Add a poll with description.

b) You can give maximum of five options.

c) You are provided with form and to date and click submit.

Edit Poll :

a) Edit any poll listed in the page.

b) Make changes and click submit.


Added Poll Questions:

View all the added poll questions here.


Currently Displaying Polls:

a) Currently displaying polls will be listed here.

b) You are provided with block option to block any polls.

c) Statistics options to view the stats of that poll.

User Rated Polls :

Provided with statistics option to view the stats.


Delete Poll Questions :

You are provided with delete option to delete any expired or unwanted poll questions.
